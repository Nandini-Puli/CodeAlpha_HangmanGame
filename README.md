# Hangman Game

A beginner-friendly text-based Hangman game developed using Python. The game randomly selects a word from a predefined list, and the player must guess the word one letter at a time before running out of attempts.

## Features
- Random word selection from a list of 5 words
- Maximum of 6 incorrect guesses
- Input validation for single-letter guesses
- Prevents duplicate guesses
- Displays correctly guessed letters and hidden letters
- Win and Game Over messages

## Technologies Used
- Python
- random module

## Concepts Practiced
- Lists
- Strings
- Loops (`while`, `for`)
- Conditional statements (`if-else`)
- User input handling
- Random number generation

This project is ideal for Python beginners who want to practice fundamental programming concepts through a fun console-based game.
